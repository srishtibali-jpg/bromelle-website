# Bromelle final Git update

This package is prepared for the existing `bromelle-website` GitHub repository.

## Included

- Updated `index.html`
- Updated `styles.css`
- Updated `script.js`
- Seven optimized WebP product images inside `assets/`

## Products

1. Peanut Butter Cookies
2. Chocochip Cookies
3. Suji Nut Tea Cake
4. Suji Fresh Fruit Cake
5. Suji Mango Cake
6. Chocochip Muffin
7. Blueberry Muffin

Suji Tuti Fruti Cake is intentionally excluded for now.

## Features

- Jaggery / Dates selector for every product
- Pack of 4 / Pack of 6 selector for muffins
- Automatic price changes
- Cart keeps each sweetener and pack-size combination separate
- WhatsApp checkout to `919999083290`
- Product selections appear in the WhatsApp order message
- Optimized images for faster loading

## Upload instructions

1. Extract this ZIP.
2. Upload `index.html`, `styles.css`, and `script.js` to the repository root.
3. Upload all seven `.webp` files into the existing `assets` folder.
4. Do not delete the existing `hero-bakes.svg` or `mom-baker.svg` files.
5. Commit the changes to the branch connected to Vercel, normally `main`.
6. Vercel should deploy the update automatically.
